# VectorOps CI/CD Pipeline

A DevSecOps portfolio project demonstrating Azure DevOps CI/CD for a Python application with automated linting, unit testing, security scanning, and dependency vulnerability scanning.

## What This Project Demonstrates

- Azure DevOps Pipelines
- Python application testing
- Ruff linting
- Pytest unit testing
- Bandit security scanning
- pip-audit dependency scanning
- GitHub portfolio documentation

## Pipeline Flow

Code Commit → Linting → Unit Tests → Security Scan → Dependency Scan → Build Validation

## Tools Used

- Azure DevOps
- GitHub
- Python
- Ruff
- Pytest
- Bandit
- pip-audit

## Security Checks

This pipeline includes:

- Static code security scanning with Bandit
- Dependency vulnerability scanning with pip-audit
- Automated quality checks with Ruff
